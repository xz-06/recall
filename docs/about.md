# 关于

这是一个专属于我们的回忆记录系统。

## 功能特点

- 📝 **记录美好时光** - 轻松记录每一个重要时刻
- 🖼️ **上传和展示照片** - 保存珍贵的照片回忆
- 📅 **按时间线查看** - 按时间顺序回顾所有回忆
- 💜 **温馨的界面设计** - Material主题，美观易用

## 技术栈

- **MkDocs** - 静态站点生成工具
- **Material主题** - 美观的界面设计
- **Markdown** - 简单易用的内容格式

## 使用说明

### 添加回忆

1. 访问上传页面
2. 填写回忆信息（标题、日期、内容）
3. 上传相关图片（可选）
4. 点击保存

### 查看回忆

- 在导航栏中选择"回忆录"
- 可以按时间或主题查看所有回忆
